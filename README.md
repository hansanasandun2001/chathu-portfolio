# Chathurangi Weerasingha Portfolio (React)

## Local Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Create a new repository on GitHub (for example, `chathu-portfolio`).
2. Commit and push this project to your repository.
3. Run the deploy script:

```bash
npm run deploy
```

4. In GitHub, go to **Settings** -> **Pages**.
5. Select **Deploy from a branch** and choose the `gh-pages` branch.
6. Save. Your site will be available at:

```
https://hansanasandun2001.github.io/chathu-portfolio
```

## Notes

- Keep all asset paths relative so GitHub Pages works correctly.
- Add your profile image at `src/assets/profile.jpg`.
